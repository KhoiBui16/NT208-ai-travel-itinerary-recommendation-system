## 🚀 Hướng Dẫn Cài Đặt

### Bước 1️⃣: Clone Repository

Sao chép repository này về máy tính của bạn:

```bash
git clone https://github.com/KhoiBui16/NT208-ai-travel-itinerary-recommendation-system.git
cd NT208-ai-travel-itinerary-recommendation-system
```

### Bước 2️⃣: Cài Đặt Dependencies

Cài đặt tất cả các thư viện cần thiết:

```bash
npm install
```

**⏳ Lưu ý:** Bước này có thể mất vài phút tùy thuộc vào tốc độ internet của bạn.

### Bước 3️⃣: Cấu Hình Environment (Tuỳ chọn)

Nếu dự án yêu cầu các biến môi trường, tạo file `.env` trong thư mục gốc:

```bash
cp .env.example .env
```

Sau đó, chỉnh sửa file `.env` với các giá trị cần thiết (nếu có).

---

## ▶️ Hướng Dẫn Chạy

### 🎨 Chạy Ứng Dụng Web (Development Mode)

Để chạy ứng dụng ở chế độ phát triển với hot-reload:

```bash
npm run dev
```

**Hoặc** sử dụng lệnh:

```bash
npm start
```
### 📱 Truy Cập Ứng Dụng

Sau khi chạy lệnh trên, ứng dụng sẽ khả dụng tại:

- 🌐 **Local URL:** `http://localhost:5173/`
- 🔗 **Network URL:** Xem trong terminal output

**Gợi ý:** Nhấn `Ctrl + C` (hoặc `Cmd + C` trên Mac) để dừng server.

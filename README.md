## 🚀 Hướng Dẫn Cài Đặt

### 1. Clone Repository

```bash
git clone https://github.com/KhoiBui16/NT208-ai-travel-itinerary-recommendation-system.git
cd NT208-ai-travel-itinerary-recommendation-system
```

### 2. Tạo Virtual Environment

```bash
node -v
npm -v
```

### 3. Cài Đặt Dependencies

```bash
npm install
```

---

## ▶️ Hướng Dẫn Chạy

### 🎨 Chạy Giao Diện Web (Demo)

```bash
npm run dev
```
Hoặc
```bash
npm start
```

Truy cập giao diện tại: `http://localhost:5173/` hoặc link Gradio Share được cung cấp.
